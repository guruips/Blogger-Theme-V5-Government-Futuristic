## Tips Menggunakan Warna

1. **Gunakan Navy Blue sebagai primary** - Warna utama untuk navigasi, CTA, dan interactive elements
2. **Gunakan Gold untuk highlights** - Accent warna untuk menarik perhatian pada elemen penting
3. **Gunakan Teal untuk secondary actions** - Warna pelengkap untuk badges, labels, status
4. **Pertahankan 80-20 ratio** - 80% neutral colors, 20% accent colors
5. **Test dengan colorblind simulator** - Pastikan design tetap terlihat untuk semua orang
